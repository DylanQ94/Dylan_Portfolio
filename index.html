Automation Engineer - Self-employed

October 13, 2025 - Present

As a self-employed Automation Engineer, I decided to work independently while dedicating focused time to my professional transition into Data Engineering. During this period, I have provided automation services for industrial environments, including PLC programming for electrical power transfer control between the plant power supply and the utility grid, PLC and HMI development for plastic roll drilling and cutting machinery, and the development of an automated reporting system using Python to extract data from a SQL Server 2012 database and generate process reports with pandas and Matplotlib.

These projects have allowed me to continue strengthening my automation background while applying programming, data analysis, database design, and problem-solving skills to real industrial use cases.

Process Engineer - Ajover

March 03, 2023 - October 03, 2025
